== Description ==

Email creation plugin - allows you to select specific articles to be included into an email template.

Beta release. More themes coming soon along with a website and tutorial.

Things left to do:

- Deleting emails
- Pagination
- Sorting emails by date

== Installation ==

1. Upload `wp-emaily` folder to the `/wp-content/plugins/` directory
1. Activate the plugin through the 'Plugins' menu in WordPress
1. Set the permissions for `emails` folder to `777`
